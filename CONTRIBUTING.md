# Contributing Content

Contributions must keep this repository content-only. Application logic,
scripts, API handlers, build configuration, and UI code belong in the frontend
repository.

Before submitting content or media, confirm that:

- you created it or have authority to contribute it;
- it contains no secrets, credentials, customer data, or private information;
- it does not misuse a trademark, likeness, or other third-party right;
- any non-CC0 asset is documented in `ASSET-LICENSES.md` with its source,
  creator, license, and required attribution.

Unless you clearly identify different compatible terms in the contribution and
in `ASSET-LICENSES.md`, you affirm that you have the necessary rights and apply
CC0 1.0 Universal to your contribution. Do not submit material that cannot be
redistributed for commercial and non-commercial use.
